## Installation

<br>

### 01. Quick start

---

- Next.js는 아래의 명령어를 통해 셋업을 진행할 수 있다.

```Terminal
<!-- npm -->
    npx create-next-app@latest my-app --yes
    cd my-app
    npm run dev
```

- --yes 옵션은 모든 질문을 건너뛰고, 저장된 설정 또는 기본값으로 프로젝트를 생성한다.
  이 기본 설정에는 TypeScript, Tailwind, ESLint, App Router, Turbopack이 포함되며,
  import alias는 `@/\*` 로 설정된다.

<br>

### 02. System requirements

---

- Next.js의 최소 node 버전은 `20.9`이다.
- 운영 체제는 아래와 같이 지원한다.
  - macOS
  - Windows (including WSL)
  - Linux.

<br>

### 03. Supported browsers

---

- Next.js는 별도의 설정 없이 최신 브라우저를 지원한다.
  - Chrome 111+
  - Edge 111+
  - Firefox 111+
  - Safari 16.4+

<br>

### 04. Create with the CLI

---

- Next.js 앱을 가장 빠르게 생성하는 방법은 `create-next-app` 명령어를 사용하는 것이다. 이 명령어는 모든 설정을 자동으로 셋업한다.

```
npx create-next-app@latest
```
